# Career Guidance Frontend

React + TypeScript + Vite frontend for the Career Guidance Web App.

## Quick Start

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`

## Configuration

Set `VITE_API_BASE_URL` in `.env` to point to backend API (default: http://localhost:3000/api)

## Features

- JWT authentication
- Assessment flow (Aptitude, Values, Personal tests)
- AI-powered chat interface
- Assessment history
- Multi-language support
- Responsive UI with Tailwind CSS
